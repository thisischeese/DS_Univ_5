4. 결론
* 숭실대는 자신과 대학 규모, 상권 규모가 모두 비슷한 대학들 중 업종 개수와 매출액이 모두 매우 낮은 편임
* 숭실대는 자신과 대학 규모가 비슷한 대학들과 비교했을 때 상권 규모의 크기에 관계없이 음식점 업종 다양성이 부족함
* 특히 일식/중식 업종의 비율은 낮고 커피/베이커리 업종의 비율이 높으므로 일식/중식, 기타세계음식을 타겟으로 창업 하는 것을 추천함

### 3. Appendix
* 한계점
